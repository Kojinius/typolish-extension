## 説明文（英語・詳細）

Typolish URL Picker is a companion extension for Typolish (typolish.com), a visual proofing and annotation tool.

**Features:**
- View all open tabs in a clean list with favicons
- Search tabs by title or URL
- Select one or multiple tabs
- Send URLs to Typolish with one click
- Single URL: auto-fills the capture input field
- Multiple URLs: batch captures all selected pages as proofs

**How to use:**
1. Open Typolish (typolish.com) in any tab
2. Click the extension icon
3. Select the tab(s) you want to capture
4. Click "Send"
5. Screenshots are automatically added to your project

**Privacy:**
This extension only accesses tab URLs to display them in the popup. No data is sent to any server except Typolish itself (typolish.com). No tracking. No analytics.

## 説明文（日本語）

Typolish（typolish.com）のコンパニオン拡張機能です。

**機能:**
- 開いているタブをファビコン付きで一覧表示
- タイトル・URLで検索
- 1つまたは複数のタブを選択
- ワンクリックでTypolishにURL送信
- 1URL: キャプチャ入力欄に自動セット
- 複数URL: 一括スクリーンショット→Proofとして自動追加

**使い方:**
1. Typolish（typolish.com）をどこかのタブで開く
2. 拡張機能アイコンをクリック
3. キャプチャしたいタブを選択
4. 「送信」をクリック
5. スクリーンショットが自動でプロジェクトに追加される
